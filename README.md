# Fastra

Fastra is an Expo app for personal finance tracking.

## Get started

1. Install dependencies

```bash
npm install
```

2. Start the app

```bash
npx expo start
```

The Expo dev server lets you open the project in a development build, Android emulator, iOS simulator, or Expo Go when supported.

## Project structure

App routes live in the `app` directory and use Expo Router file-based routing.

## Scripts

- `npm run start`
- `npm run android`
- `npm run ios`
- `npm run web`
- `npm run lint`
